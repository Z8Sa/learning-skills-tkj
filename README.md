# 📚 TKJ Learning Menu

Program Python sederhana untuk membantu siswa **Teknik Komputer dan Jaringan (TKJ)** memilih materi pembelajaran dan mendapatkan link belajar sesuai topik yang dipilih.

---

## ✨ Fitur

* Menu materi TKJ berbasis terminal (CLI)
* Navigasi menggunakan `if`, `elif`, dan `else`
* Menampilkan daftar submateri berdasarkan pilihan pengguna
* Menyediakan link belajar untuk setiap materi
* Cocok untuk latihan dasar Python dan logika percabangan

## 🚀 Cara Menjalankan

Pastikan Python sudah terinstal.

Cek versi Python:

```bash
python --version
```

Jalankan program:

```bash
python menu.py
```
